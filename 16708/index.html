[Opera 套件] Opera 地表最靚瀏覽器 絕美字型無可挑剔

2010-12-18

Browser Opera Opera Font

你是否羨慕麥金塔漂亮的文字渲染（Rendering）效果，現在你的Opera也可以輕鬆辦到。沒錯，就在微軟的作業系統中。「甚麼，一向給中文使用者印象非常不好的Opera中文顯示效果，也可以辦到像麥金塔一樣的文字渲染效果！？」不相信嗎？接下去看你就知道我不是在開玩笑。（如果不喜歡安裝延伸套件［這裡］也有其他方法）

早期，Opera在處理中文（或東方文字）時的顯示效果非常糟糕，這導致很多中文使用者對Opera留下很不好的印象。但這些早已成為往事了。透過Opera 11版的最新功能－－延伸套件（Extensions）和CSS技術的加持下，現在Opera可謂是「地表最靚的瀏覽器」了。下圖就是完成設定後的最終成效。

準備動作：
若無意外，當你安裝一個全新的Opera瀏覽器時，你會看到下圖這樣的文字效果。標準的新細明體字型，很糟糕的鋸齒效果。對看習慣的使用者來說可能會覺得沒甚麼了不起的，但事實上你不需要這樣折磨你的眼睛。

預備工作：

[瀏覽器] CSS 助你向難看的 Opera 字型說掰掰

參考上述的文章，先利用CSS檔案和微軟正黑體（或其他你喜歡的字型）來取代Opera預設的字型。為什麼要使用這樣的方式呢？因為這樣的方式在設定上最簡單，也最沒有副作用。當你哪天想要換回原本的樣子時，隨時可以取消套用CSS的效果。這就是為什麼我使用CSS的方式來設定字型。完成上述的動作後，Opera的字型效果就會變身成下圖的樣子了。好看很多，不是嗎 XD

安裝套件 開始變身
拜Opera 11最新的功能－－延伸套件（Extensions）－－所賜，使用者現在可以輕鬆的透過安裝套件的方式來增加瀏覽器的功能。這次要讓Opera變身成為「地表最靚瀏覽器」的延伸套件就是這個名為「Mac OSX Font Rendering」的套件。廢話不說，馬上開始吧！

  1. 先開啟［O 選單］，點選［延伸套件］子選單的［取得延伸套件］項目

    2. 進入Opera延伸套件網頁後，輸入「Mac OSX Font Rendering」後點選搜尋按鈕。
    3. 找到我們的目標套件後，點選安裝按鈕。
    > 提示：

    你無法安裝嗎？這是因為安裝延伸套件是Opera最新版 11版的全新功能。如果你還沒更新Opera，就點［這裡］下載安裝最新版的Opera吧！ 
    4. 完成安裝套件後，點選［重新整理］按鈕讓網頁重新載入。這樣美美的字型效果馬上就呈現在你的眼前囉！
    就是不想更新！！
    由於Opera 11有很多創新性的功能，對於一些使用者來說可能還需要一段時間適應或觀望。但是這樣就無法使用這個強化顯示效果的功能也太可惜了。但是別擔心，這個延伸套件的原理是使用JavaScript的方式來達到渲染效果的。所以如果你現階段真的不想更新到Opera 11，但又很喜歡這樣的渲染效果的話，你可以考慮自行套用JS檔案到你目前的Opera中。

  2. 輸入［http://userstyles.org/styles/31924］網址進入提供文字渲染效果的JavaScript網站。選擇你的瀏覽器並點選［Install as User JS］按鈕。

    2. 網頁載入這個渲染效果的JavaScript程式碼，把這些程式碼複製到一個新的JS檔案中並把它跟你目前的Opera瀏覽器作功能連結。
    總結
    自己設定JS的方式固然可行，但終究不是很方便。如果擔心設定的不好，倒不如更新到最新版的Opera，再安裝套件。這樣會簡單很多喔！
