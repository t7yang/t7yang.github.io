[社交工具] 還在用 WLM？改用 Miranda 吧（下載安裝篇）

2010-11-28

Instant Messenger

還在因為Windows Live Messenger的肥大、消耗系統資源、和常常當不停等原因而煩惱嗎？別以為這是無可奈何的事！使用者可是有選擇的權利，改用更好用、更簡單的聊天工具－－Miranda IM吧。本文是〔還在用 WLM，改用 Miranda 吧！〕系列文章之〔下載安裝篇〕，來看看下載Miranda要先做哪些事。

我在〔還在抱怨 WLM 嗎？改用 Miranda IM 吧！〕一文已經說過，Miranda IM（以下簡稱為MI）擁有多個超越WLM的優點。但是因為開放式的設計，原始未經調整過MI簡直不是人用 XD。所以這篇文章的目的就是告訴讀者下載安裝MI需要準備些甚麼，因為對一個未曾接觸過MI這樣自由的程式時，一開始的起步會非常艱難。如果有人按部就班帶著走的話，就可以輕鬆很多。廢話不多，馬上開始吧！

下載與免安裝
我的原則還是一樣，「可以不用安裝的軟體，我通常不會選擇安裝」。再加上我正是要提倡MI的可攜式優點，所這次的安裝選擇還是免安裝。

不知道如何管理免安裝軟體嗎？何不參考：

  • RocketDock 如何免安裝與設定
  • RocketDock+Stack Docklet 打造免安裝軟體庫

  1. 首先先到MI的官方網站去下載可以在你的系統執行的版本，一般人應該都是選擇「XP、Vista、7」的那個版本。

    2. 因為要做成可攜式的版本，所以我們不需要安裝，只需要把安裝檔案解壓縮就可以了。（你不會解壓縮嗎？請下載7zip）
    3. 然後把圖中的〔$PlUGDIR〕這個資料夾刪除。這樣就完成可攜式化的處理了，很簡單吧。
    安裝中文化
    因為MI預設是英文介面，對於多數人來說中文介面還是比較親切一些。所以接下來的步驟就是教讀者如何把中文化檔案下載下來，並安裝上去。

  2. 開啟MI的附加元件網頁，在那裏可以找到中文化的檔案。

    2. 在網頁的右下方，點選〔Localization〕連結進入翻譯區。
    3. 在眾多翻譯檔案中找到〔繁體中文語言套件〕，並下載它。
    4. 把壓縮檔內的檔案拖曳到MI的資料夾當中，這樣開啟後就是中文介面了。
    新增帳號
    完成中文化後，我們就要來新增我們的聊天帳號了。別忘了，比起WLM一次只能開啟一個帳號，MI可是大方提供使用者在每個協定中同時開啟多個帳號。接下來我們就來看看，在MI中要如何新增帳號。以下我會給讀者演示一次如何新增MSN和Google Talk的即時通訊帳號，其他協定的帳號也大同小異，讀者們就自行鑽研吧。

  3. 首先當然是開啟MI主程式了。

    2. 因為我們是第一次開啟MI，所以MI會要求你先生成一個設定檔。
    3. 接著點開〔功能表〕，然後點選〔帳號〕。點選左下方的〔＋〕。
    4. 我們先演示一次新增MSN帳號的過程，名稱隨便填，協定選擇〔MSN〕。
    5. 輸入帳號和密碼就完成帳號新增了。
    6. 接下來是演示如何新增Google Talk帳號，一樣是點〔＋〕號。注意！這次要選〔Jabber〕協定。
    7. 在〔類型〕那裏選擇〔Google Talk〕，再依序輸入帳號和密碼即可。
    新式主題與分頁視窗
    相信讀者已經有開啟MI的經驗了，你一定會覺得MI的那個主視窗很小又難看（我是這樣想的）。所以我建議使用者還是選擇啟用新式的聯絡清單（Modern Contact List），因為我在之後的文章（外觀主題篇）中還會介紹如何安裝新式聯絡清單的其他介面主題。還有，啟用分頁視窗的外掛程式可以幫助管理聊天視窗，不會因為開啟太多聊天視窗導致系統工作列被占滿。

  4. 點開〔功能表〕後點選〔選項〕。

    2. 在左邊的功能選取區中，點選〔插件〕（不知道為什麼譯者要這樣翻，明明可以翻成外掛）。
    3. 在〔Modern Contact List〕和〔TabSRMM〕這樣個外掛前打勾。
    4. 最後在系統匣區點選MI的圖示退出，重新啟動才能套用變更。
    最後提醒：
    目前這樣的調整的MI，還是遠遠沒有達到很好用的階段，起碼目前沒有音效的通知還有更漂亮的主題。這些調整設定都會在後續的其他文章給你帶來。畢竟想要把所有內容都塞到一篇文章，是不切實際，讀者也會看得不耐煩。
