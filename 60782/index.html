[系統清理] FCleaner v1.3.1.621 (中文化)

2010-07-15

中文化 Cleaner Defraggler FCleaner Gmail ViDown Web HardDisk

FCleaner 是針對 Windows 作業系統的清理工具，它具備清理系統的記錄檔、暫存檔以及無效的登錄檔來最佳化 Windows 的執行力。雖然 FCleaner 的功能非常好用，但是卻以免費軟體授權給一般使用者來使用。所以如果想要定時的清理系統，FCleaner 應該是一個很好的選擇。

版本說明
FCleaner v1.3.1.621 Updated on Jul 12th, 2010

  1. 新增自動清除指定天數的資源回收筒的已刪除項目
  2. 新增顯示資源回收筒中分析或清理中的詳細資訊
  3. 改善資源回收筒的清理功能
  4. 新增字串 ‘lbl_RecycleBin’、’lbl_RecycleBin2’ 跟 ‘msg_RecycleBinEx’ 到語言檔
  5. 一些微調
